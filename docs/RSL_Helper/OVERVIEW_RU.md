# RSL Helper V6 — Обзор проекта

## Описание
**RSL Helper V6** — это сторонняя утилита для игры **Raid: Shadow Legends** (популярная мобильная/ПК RPG от Plarium). Программа позволяет игрокам просматривать и управлять игровыми ассетами локально на компьютере.

## Основные возможности
- Просмотр портретов и изображений чемпионов
- Просмотр экипировки (оружие, шлем, нагрудник, перчатки, ботинки, щит)
- Просмотр навыков и умений чемпионов
- Просмотр мастерств (большие и малые иконки)
- Управление реликвиями и драгоценными камнями
- Поддержка нескольких версий игровых данных (10.41.0, 10.60.0, 11.20.0 и др.)
- Автоматическое обновление через встроенный апдейтер
- Извлечение ассетов из Unity3D бандлов игры

## Структура проекта

### Основные исполняемые файлы
- **RSLHelper.exe** — главное приложение
- **RSLHelper_Updater.exe** — компонент автообновления
- **rslhelper.dll** / **rslhelper_v80.dll** — основные библиотеки (логика парсинга игровых данных)

### Директории с ассетами
- **aPictures/** (523+ файлов) — Изображения экипировки по слотам (ботинки, нагрудник, перчатки, шлем, щит, оружие), пронумерованные по ID чемпиона
- **bPictures/** — Иконки мастерств/способностей (большие и малые версии) с рамками редкости (редкие, эпические, легендарные)
- **cPictures/** (1123+ файлов) — Портреты/миниатюры чемпионов
- **iPictures/** — Различные игровые иконки:
  - `bm/` — Иконки боевых механик
  - `event/` — Изображения событий
  - `relict/` — Ассеты системы реликвий (драгоценные камни, иконки реликвий, камни реликвий, миниатюры)
  - `rv/` — Дополнительные игровые ассеты
- **Skills/** (3530+ файлов) — Иконки навыков/умений чемпионов (несколько уровней на каждый навык)
- **UserFav/** — Хранилище избранных элементов пользователя

### Файлы данных
- **ci.dat** — Файл конфигурации/данных
- **rslSim.dat** — Данные симуляции
- **StandardJson.hsf** — Конфигурация JSON-правил
- **SQLITE3.DLL** — Поддержка базы данных SQLite
- **libeay32.dll** / **ssleay32.dll** — Библиотеки шифрования OpenSSL
- **FinalSound.wav** — Звуковой файл

### Файлы логов
- **_tmp_bundle_debug.log** — Отладочный лог обработки бандлов ассетов
- **UnzipErrorLog.Bytes** — Лог ошибок распаковки

## Принцип работы
Программа считывает игровые бандлы ассетов из установки Plarium Play Raid: Shadow Legends, обрабатывает Unity3D бандлы (например, `HeroAvatarsLocal`, `FuseAvatarsLocal`), извлекает данные чемпионов и рендерит атласы изображений.

## Назначение
Это **средство просмотра/извлечения ассетов Raid: Shadow Legends**, которое позволяет игрокам просматривать портреты чемпионов, экипировку, навыки, мастерства, реликвии и другие игровые ассеты локально на компьютере, что удобно для планирования состава команды или детального изучения игровых ресурсов.
